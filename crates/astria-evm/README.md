# Symphony

Symphony is an EVM built in Rust to run on top of the [Astria Shared Sequencer Network]((https://github.com/astriaorg/astria)) based on the [Reth EVM](https://github.com/paradigmxyz/reth).

## Installation
Pull the repo:
`git clone git@github.com:astriaorg/astria.git`
`cd astria`

To install locally:
`cargo install --locked --path crates/astria-evm --bin symphony`
To run: `symphony node`
